# Design Brief — Mimo Public Funnel

## Overview
Mimo is a bilingual (Arabic / English) subscription platform for SMB owners in Egypt and the GCC. It delivers a monthly **Strategic Content System** — a complete content operating system that replaces the need for an in-house content team.

## Goals
- Establish Mimo as the definitive content operating system for MENA SMBs
- Communicate the value of the system in plain business language, not tool language
- Convert visitors to subscribers through clarity and credibility, not hype
- Support bilingual (AR/EN) switching without layout disruption

## Target Audience
- SMB founders and marketing leads in Egypt, Saudi Arabia, UAE, and the GCC
- Businesses with 5–50 employees that need consistent content output but lack a content team
- Owners currently spending hours per week on scattered, low-quality content

## Branded Terminology (use these exact phrases — no AI language)
| Platform term | Never say |
|---|---|
| Strategic Content System | AI content tool / AI writer |
| Brand Diagnosis | AI audit / AI analysis |
| Mini Brand Book | AI-generated brand guide |
| Strategic Content Workflow | AI workflow / automation |
| Content Operating System | AI platform |
| Monthly Content Engine | AI content calendar |

## Tone & Voice
- Confident, direct, business-first
- Arabic copy: Modern Standard with Gulf warmth, not formal Egyptian bureaucratic
- English copy: concise, zero jargon
- Never: "magic," "AI-powered," sparkle emoji, purple gradients, "GPT"

## Visual System — Midnight Glow
- **Surface**: light (#f7f8fa base), not white-white — warm off-white
- **Primary**: deep navy (#071b2e) for headings and structural elements
- **Accent**: teal (#1fa090) for CTAs and interactive elements
- **Glow**: cyan (#22d3ee) — atmospheric only, hero panels, never UI chrome
- Mobile-first. Always. Desktop is an enhancement.

## Key Pages (Public Funnel)
1. **Hero / Landing** — what the system does and who it's for
2. **How It Works** — the Strategic Content Workflow, 3-step visual
3. **What You Get** — deliverable breakdown (Brand Diagnosis, Mini Brand Book, etc.)
4. **Pricing** — single subscription tier, clear value
5. **Social Proof** — client outcomes, not testimonials with stars
6. **Subscribe** — low-friction email/phone capture → payment

## Constraints
- WCAG 2.1 AA minimum
- Lighthouse performance ≥ 90 on mobile 4G
- RTL layout must work without a separate codebase — CSS logical properties only
- No analytics or tracking on the subscribe form until user opts in

## Success Metrics
| Metric | Target |
|---|---|
| Landing → Subscribe conversion | ≥ 5 % |
| Bounce rate | ≤ 40 % |
| Time to subscribe (from first visit) | ≤ 2 min |
| Mobile Lighthouse performance | ≥ 90 |
| Accessibility score | 100 |
